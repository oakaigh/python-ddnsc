# python-ddns

## Installation
```sh
make deploy
```
```sh
systemctl status ddnsc-daemon
```

## Examples
```json
{
    "volatile": true,
    "filter": {
        "conf": {
            "interfaces": ["eth0"]
        }
    },
    "database": {
        "name": "cloudflare",
        "conf": {
            "conf": {
                "email": "user@example.com",
                "token": "c2547eb745079dac9320b638f5e225cf483cc5cfdda41"
            },
            "zone": "example.com",
            "name": "@"
        }
    }
}
```
